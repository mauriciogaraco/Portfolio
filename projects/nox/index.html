Nox Cookies Bar

A full-stack e-commerce project with a built-in web editor, a Stripe payment gateway, and a powerful dashboard for data.

React.js Nodejs Stripe PostGreSQL
Live ↗

Nox Cookies Bar is a full-stack project designed to boost sales for Nox. It's more than just a modern e-commerce site; it includes a Stripe payment gateway and an advanced dashboard with charts and metrics for data analysis. The project also features a built-in web editor using GrapesJS, allowing for easy drag-and-drop customization.
